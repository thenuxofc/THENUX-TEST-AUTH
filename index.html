📘 How it works

        
This page uses the thenux-otp.js SDK from your THENUX installation.

        The SDK automatically handles:

        
    
            
  • Displaying a login modal (Google or email/password)
    • 
            
  • Exchanging Firebase credentials for a THENUX JWT
    • 
            
  • Showing OTP input if the user has 2FA enabled
    • 
            
  • Returning the final token and user data via the onSuccess callback
    • 
        

        
After login, you can use the returned JWT to authenticate requests to your own backend.

    


    
    

    

    

        
🔧 Manual API integration (alternative)

        
If you prefer to build your own UI, you can directly call the REST endpoints:

        
// 1. Start OTP session
fetch('https://thenux-auth.netlify.app/.netlify/functions/send-otp', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'x-thenux-key': 'YOUR_API_KEY' },
    body: JSON.stringify({ email: 'user@example.com' })
})
.then(r => r.json())
.then(data => {
    const otpSessionToken = data.otpSessionToken;
    // 2. Verify code (user input)
    return fetch('https://thenux-auth.netlify.app/.netlify/functions/verify-otp', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json', 'x-thenux-key': 'YOUR_API_KEY' },
        body: JSON.stringify({ code: '123456', otpSessionToken })
    });
})
.then(r => r.json())
.then(verifyData => {
    console.log('JWT token:', verifyData.tokens.accessToken);
});
